# 患者自助挂号系统

一个功能完整的医院自助服务微信小程序，提供挂号、缴费、查询等多项便民服务。

## 功能模块

### 1. 当日挂号
- 科室选择
- 医生列表展示
- 实时号源查询
- 在线挂号支付

### 2. 预检挂号
- 患者信息登记
- 症状描述
- 紧急预检服务

### 3. 挂号预约
- 未来14天预约
- 科室医生选择
- 时段预约
- 预约管理

### 4. 预约取号
- 预约单查询
- 在线取号
- 订单管理

### 5. 缴费
- 待缴费查询
- 多项目选择
- 在线支付
- 缴费记录

### 6. 自助查询
- 挂号记录查询
- 缴费记录查询
- 检查报告查看

### 7. 入院登记
- 患者信息填写
- 科室床位选择
- 紧急联系人登记

### 8. 病历打印
- 病历记录查询
- 多份选择打印
- 打印管理

## 技术栈

- **框架**: 微信小程序原生开发
- **渲染引擎**: Skyline
- **组件框架**: Glass-easel
- **UI设计**: 现代化渐变色彩设计
- **交互**: 流畅的用户体验

## 项目结构

```
miniprogram-1/
├── pages/                    # 页面目录
│   ├── index/               # 主页
│   ├── today-register/      # 当日挂号
│   ├── pre-check/           # 预检挂号
│   ├── appointment/         # 挂号预约
│   ├── take-number/         # 预约取号
│   ├── payment/             # 缴费
│   ├── inquiry/             # 自助查询
│   ├── admission/           # 入院登记
│   └── print/               # 病历打印
├── images/                  # 图片资源
├── app.js                   # 小程序逻辑
├── app.json                 # 小程序配置
├── app.wxss                 # 全局样式
└── README.md               # 项目说明
```

## 使用说明

### 开发环境
1. 安装微信开发者工具
2. 导入项目目录
3. 编译运行

### 配置说明
- 项目使用 Skyline 渲染引擎
- 支持自定义导航栏
- 启用懒加载优化

### 数据说明
当前版本使用模拟数据，实际使用时需要：
1. 对接医院HIS系统API
2. 实现真实的支付接口
3. 添加用户认证功能
4. 完善数据安全措施

## 界面特色

- **渐变色彩**: 每个功能模块使用独特的渐变色彩
