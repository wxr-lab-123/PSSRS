========================================
    患者自助挂号系统 - 项目说明
========================================

项目名称：患者自助挂号系统
开发时间：2024-11-03
版本号：V1.0
开发框架：微信小程序（Skyline渲染引擎）

========================================
        功能模块列表（共8个）
========================================

1. 当日挂号 (today-register)
   - 科室选择
   - 医生列表
   - 实时挂号

2. 预检挂号 (pre-check)
   - 信息登记
   - 症状描述
   - 紧急服务

3. 挂号预约 (appointment)
   - 日期选择
   - 科室医生
   - 时段预约

4. 预约取号 (take-number)
   - 订单查询
   - 在线取号

5. 缴费 (payment)
   - 费用查询
   - 在线支付
   - 缴费记录

6. 自助查询 (inquiry)
   - 挂号记录
   - 缴费记录
   - 检查报告

7. 入院登记 (admission)
   - 患者信息
   - 科室床位
   - 联系人

8. 病历打印 (print)
   - 病历查询
   - 批量打印

========================================
        技术特点
========================================

✓ 使用 Skyline 渲染引擎，性能更优
✓ Glass-easel 组件框架，开发更高效
✓ 自定义导航栏，界面更美观
✓ 渐变色彩设计，视觉效果出色
✓ 响应式交互，用户体验流畅
✓ 模块化开发，代码结构清晰

========================================
        数据说明
========================================

当前版本：使用模拟数据
实际部署：需要对接医院HIS系统

模拟数据包括：
- 科室信息（8个科室）
- 医生信息（含职称、擅长、价格）
- 挂号记录
- 缴费记录
- 检查报告
- 病历记录

========================================
        使用步骤
========================================

2. 导入项目目录

4. 在模拟器或真机预览

详细使用说明请查看：使用指南.md
